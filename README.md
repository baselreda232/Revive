# Revive

Revive is a furniture e-commerce and services app with a React/Vite frontend and an Express/MongoDB backend.

## Project Structure

- `src/` - React frontend source
- `reviveeB/` - Express backend source
- `public/` - frontend static assets
- `.env.example` - frontend environment template
- `reviveeB/.env.example` - backend environment template

`dist/` is generated by the frontend build and is ignored by Git.

## Frontend

```bash
npm install
npm run dev
```

Create a local `.env` from `.env.example` when needed:

```bash
VITE_API_BASE_URL=http://localhost:3000
```

## Backend

```bash
cd reviveeB
npm install
npm run dev
```

Create `reviveeB/.env` from `reviveeB/.env.example` and fill in the MongoDB, token, email, admin, and client URL values.

## Checks

```bash
npm run lint
npm run build
cd reviveeB
npm audit --audit-level=moderate
```
